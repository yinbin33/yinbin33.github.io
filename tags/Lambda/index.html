Lambda

Lambda表达式

1. 基本特点

  • 代码清晰
  • 改善代码臃肿
  • 函数类型编程
  • 兼容接口升级

2. 实现方式

  • Lambda语法

  • 方法引用

  • 接口default方法实现

  • @FunctionalInterface接口

    1. 类型是接口类型，而不是注释类型、枚举或类。
    2. 该注解需满足函数接口^1的要求。

3. 局限性

  • 单一抽象方法
  • Lambda调试困难
  • Stream API操作能力有限

4. 常用模式（scfp = Supplier + Consumer + Function + Predicate）

  • Supplier：提供者，形如无参有返回值
  • Consumer：消费者，形如有参无返回值
  • Function：函数，形如有参有返回值
  • Predicate：断言，有参有返回值，返回boolean
接口 参数 返回类型
Predicate T boolean
Consumer T void
Function<T, R> T R
Supplier None T
UnaryOperator T T
BinaryOperator (T,T) T

示例

  1. 【Predicate】判断一个值是否大于5

  2. 【BinaryOperator】计算两个整数之和
